# Grub-Menu

A React recreation of the classic GRUB boot menu — with a twist.  
This project showcases:
- **Coolbuntu** → a custom Ubuntu‑inspired UI I designed
- **Windows 12 UI leaks** → a parody OS interface
- **Recovery Mode** → simulated fallback environment
